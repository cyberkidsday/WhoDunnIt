## Description

CTF style "Who Dunn It" game for children

## Building and running

1. Clone repo
```
git clone https://github.com/cyberkidsday/whodunnit.git
```
2. install npm dependencies where package.json is
```
npn install
```
3. run the app
```
npm start
```
